# README #
### 更新源码请切换 GetPadavanNgSrc 为默认分支 ###
### 编译固件请切换 PadavanNgCBuild 为默认分支 ###
* 点击 ☆Star 只更新源码
* 新建 Release 更新源码和预编译工具链
* 源码更新到分支 PadavanNgMaster
* 工具链更新到 Releases
***
* 新建 Release 时的 Tag version (只能填字母和数字)即是工具链版本
* 编译脚本默认工具链版本为 Rev1 (不支持符号)
* 更新工具链时请修改编译脚本工具链版本
***
